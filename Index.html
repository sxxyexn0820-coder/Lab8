🔗 실시간 공유 설정 (Firebase)

Firebase를 연결하면 A, B, C가 동시에 같은 재고를 수정하고 실시간으로 볼 수 있습니다.
Google 계정만 있으면 완전 무료입니다. 6단계, 5분이면 완료!

  1. console.firebase.google.com → 프로젝트 추가
  2. 프로젝트 이름 입력(예: lab-inventory) → Google 애널리틱스 끄기 → 만들기
  3. 왼쪽 빌드 → Realtime Database → 데이터베이스 만들기
    위치: asia-southeast1 → 테스트 모드로 시작
  4. 왼쪽 상단 톱니바퀴(⚙️) → 프로젝트 설정
  5. "내 앱" 섹션 → </> 웹 아이콘 → 앱 이름 입력 → 앱 등록
  6. 나오는 firebaseConfig 전체 복사 → 아래 붙여넣기
📋 정확한 붙여넣기 방법
Firebase 콘솔에서 firebaseConfig 코드를 복사하면
아래처럼 생겼습니다. 전체를 그대로 복사해서 붙여넣으면 됩니다 (수정 불필요):
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "your-project.firebaseapp.com",
  databaseURL: "https://...",
  projectId: "your-project",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
